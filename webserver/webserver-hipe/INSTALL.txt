- récupérer le code sur https://gitlab.aneo.fr/tollivier/hipe

- effectuer un npm install

- faire un gulp js, puis gulp css, puis gulp sass (s'il ne connait pas le command gulp, ouvrir un nouveau cmd, si ça ne marche toujours pas, c'est qu'il y a un problème sur le package)
