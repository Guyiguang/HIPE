Detailed Description

The image needs to be in grayscale. If not it will be converted at runtime. The filter will first blur the image (for better results), then use the canny algorithm. The output of the canny algorithm will help to find the contours with the OpenCV findContours function.

See also
cv::Canny()
cv::findContours()
